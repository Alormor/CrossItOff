# Cross It Off!

This is a to-do list web app that uses Firebase for authentication and database.

Its URL is: https://todolist-bc0ca.web.app/

It has a fun and colorful design with various features like:

- **Authentication:** Users can register, login, and logout.
- **To-do list:** Users can add, edit, delete, and mark tasks as completed.
- **File upload:** Users can upload files to their tasks.
- **Admin panel:** Admins can view and delete all tasks, and delete any file.

## Project Setup

```sh
npm install
```

### Compile for Development

```sh
npm run dev
```

### Compile for Production

```sh
npm run build
```
